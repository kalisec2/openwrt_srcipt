MT3600BE 安装openclash一键脚本未加密

先下载脚本到openclash

```shell
wget https://gh-proxy.org/https://raw.githubusercontent.com/kalisec2/openwrt_srcipt/refs/heads/main/go_opneclash.sh 
```

给脚本权限

```sh
chmod +x install_openclash.sh

```

执行脚本，如果第一行执行不了。请使用第二行

```sh
./install_openclash.sh
```

换下面的语句执行

```sh
bash install_openclash.sh
```
